# Sudoxir

A parallel brute-force Sudoku solver written in Elixir.

## Requirements

Elixir (`brew install elixir` on OSX)

## To build

```
mix escript.build
```

## To run the tests

```
mix test
```

## To run the program

```
$ ./sudoxir board.txt
```
